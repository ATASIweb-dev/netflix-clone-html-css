# Netflix Clone (HTML & CSS)

A UI clone of the Netflix homepage built using pure HTML and CSS.

This project focuses on layout design, responsiveness, and visual accuracy without using JavaScript.

## Features
- Responsive layout
- Flexbox and CSS Grid usage
- Modern UI styling
- Clean and structured code

## Technologies Used
- HTML
- CSS

## Project Status
This project is currently not deployed.
A live version will be added after further improvements.
